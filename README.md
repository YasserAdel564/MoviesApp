# MoviesApp

MeOnline is Educational App For Watch Different Courses and Request it 

## Features
- [x] Kotlin
- [x] Kotlin Coroutines
- [x] Clean Architecture with MVVM
- [x] Retrofit2
- [x] Paging2
- [x] Repository Design Pattern
- [x] DI(Hilt)
- [x] View Binding
- [x] Navigation Components
- [x] Google Material Design
